# [현장 보고용 표준 행정 순화 용어 대조표]

본 문서는 현장 요원이 보고서 작성 시, 중앙관리국의 **[행정 무결성 감사]**에서 지적받지 않도록 감정적/비논리적 표현을 행정 표준 용어로 치환하는 기준을 제시한다. 아래 대조표를 위반한 보고서는 즉시 파기되며 작성자는 인지 검진 대상으로 분류된다.

| 금지 표현 (감정적/주관적) | 권장 표현 (행정적/객관적) | 적용 예시 |
| --- | --- | --- |
| "괴물이 울부짖는다" | "비정형 개체가 고주파 음성 데이터를 출력함" | "개체가 타격에 반응하여 고주파 데이터를 출력함." |
| "동료가 잡아먹혔다" | "인적 자원이 개체와 위상적으로 통합됨" | "김 요원의 신체가 개체와 위상적으로 통합되어 자원 손실 발생." |
| "지옥 같은 풍경" | "고농도 RR 지수(%) 노출로 인한 환경 해상도 열화" | "현장은 현재 심각한 환경 해상도 열화 상태임." |
| "귀신이 벽을 통과함" | "비정형 자산이 물리적 방벽을 위상적으로 무시함" | "자산이 차폐벽을 위상적으로 무시하고 이동함." |
| "너무 무섭다/공포스럽다" | "인지 부하로 인한 논리적 판단 지연" | "요원들이 인지 부하로 인해 논리적 판단 지연을 겪음." |
| "피가 쏟아진다" | "고농도 유기 점액질 유출" | "현장에서 다량의 유기 점액질 유출 확인." |

## ※ 주의사항
보고서에 "살려주세요", "신이시여" 등의 종교적/절박한 문구를 삽입하는 행위는 **[인지 무결성 훼손]**의 결정적 증거로 간주한다. 이러한 문구가 발견될 경우, 해당 보고서의 해시값은 즉시 **'오염'**으로 판정되어 시스템에서 삭제된다.
