# 도시야간안전 인프라 공단 — 세계관 및 문서 양식 분석

---

## Part 1. 문서 양식 분석

### 1-1. 전체 구조: Obsidian Vault 기반의 "가상 행정 전산망"

총 77개 마크다운 파일이 9개 상위 디렉토리로 분류되어 있으며, 현실의 공기업 사내 전산 시스템을 모사한다.

```
00. 공단_본부         → 본부 행정 (표준양식, 용어, 인사)
01. DUSI_중앙관리국   → 최상위 통제 기관 (음각, 소급말소, 감사)
02~08. (주)하청법인들  → 기능별 자회사 7개
```

각 하청법인 내부도 `[4자리 코드]. 업무영역 / 01. 세부분류 / 0001. 개별문서.md` 형태로 3단계 중첩 폴더를 사용하고 있어서, 실제 정부기관 전산 문서 분류 체계(기록물 분류 기준표)를 그대로 흉내내고 있다.

### 1-2. 문서 헤더: YAML Frontmatter의 이중 ID 시스템

모든 문서가 동일한 YAML Properties 구조를 가진다.

```yaml
영문명: "Standard Administrative Vernacular"
문서번호:            # Public ID - 하청업체에 노출
  - "관리-1001-20XX-00001"
분류번호:            # Internal DUSI ID - 하청 노출 금지
  - "관리-DOC-1001-0001"
  - <!-- 하청 노출 금지 -->
  - "DUSI-1001-01-0001"
보안등급:
  - "[등급: 심의종결-01]"
공표일자: "20XX. XX. XX."
```

`<!-- 하청 노출 금지 -->` HTML 주석이 두 ID 사이에 삽입되어, "이 문서를 보는 사람조차 이중 체계를 인지해서는 안 된다"는 세계관 내 규칙을 문서 양식 자체로 체현한다. Obsidian에서 이 주석은 렌더링되지 않으므로, 편집 모드에서만 보이는 "메타 비밀"이 된다.

### 1-3. 보안등급 체계

문서 전체에서 3단계 보안등급이 사용된다.

| 등급 | 의미 | 용도 |
|---|---|---|
| `심의종결-01` | 일반 행정 | 지침서, 양식, 카탈로그 |
| `결재보류-02` | 제한 열람 | 현장 보고서, 임상 데이터 |
| `소급말소-00` | 최고 기밀 | 소급 말소 명부 등 |

### 1-4. 문서 유형의 분류

양식 가이드에서 명시한 문서 유형별 서식 규칙은 다음과 같다.

- **서식(FORM)**: 빈 칸(`______`)과 체크박스(`(  )`)가 있는 양식
- **사례(DOC)**: 양식에 실제 데이터가 채워진 기록
- **SOP**: 절차 매뉴얼

이 3종이 쌍을 이루며, 예를 들어 `0001. 서식` 다음에 `0002. 사례`가 오는 구조로 "빈 양식 → 작성 예시"가 반복된다. 이것이 실제 관공서 업무 매뉴얼의 구성을 매우 정확하게 모사한다.

### 1-5. 문서 말미의 "감시 경고문"

모든 문서의 끝에 동일한 경고문이 반복된다.

> **[DUSI 중앙 데이터베이스 관리 통고]**
> 본 문서는 [...] 실시간으로 동기화 및 무결성이 감시되는 기록물입니다.
> [...] 예외 없이 **[소급 말소]** 및 **[행정적 재사회화]** 대상으로 강제 전환됩니다.
> *해당 문서는 시스템에 의해 추적되고 있습니다 (Current Access Node: DUSI-NODE-ADMIN).*

한국 행정 문서의 관인 날인 위치(문서 맨 아래)에 배치되어, "관인 대신 경고문"이라는 대치 효과를 만든다.

### 1-6. 서식 규격화의 디테일

양식 가이드(`0001. 공용_문서_양식_가이드`)에서 정의한 규칙들은 다음과 같다.

- 체크박스: `(  )` (소괄호 + 공백 2칸)
- 서명란: `______________________________` (언더바 30자)
- 빈 표 칸: 특수 공백 문자 `ㅤ`로 너비 유지
- 행정 용어: **[단어]**, 기술 용어: `'용어'`, 날짜: `20XX. XX. XX.`

이런 미시적 서식 규칙까지 통일한 것이 "실제로 운용되는 시스템"의 느낌을 극대화한다.

## Part 2. 세계관 분석

### 2-1. 핵심 전제

> **도시 야간 구역에 '비정형 개체'(초자연 존재)가 서식하고 있으며, 한국 정부 산하 공단이 이를 은폐하고 관리한다.**

이 세계관의 공포는 개체 자체가 아니라 **"그것을 관리하는 관료제 시스템"**에 있다. 은폐와 행정적 처리, 인력의 소모적 운용, 존재 자체의 말소가 세계관을 관통하는 핵심 동력이다.

### 2-2. 조직 구조: 공기업 + 하청의 위계

```
DUSI 중앙관리국 (9000) ─ 실질적 지배자, 모든 최종 결정권
    │
    ├── 공단 본부 (1000) ─ 행정 표준, 법령, 인사
    │
    └── 하청 법인 7개 (각각 (주) 형태)
         ├── (주)특수시설유지관리 (2000) ── 현장 조사 (탐색 + 발견)
         ├── (주)야간환경정비     (3000) ── 물리적 처리 (마모 + 세척)
         ├── (주)유성물류기획     (4000) ── 운송 + 격리
         ├── (주)바른도시건설     (5000) ── 대외 은폐 + 유령지구
         ├── (주)미래생명과학     (6000) ── 인지 의료 + 재사회화
         ├── (주)가온정밀시스템   (7000) ── 장비 + 해시 검증
         └── (주)특수자산전략분석 (8000) ── 개체 제압 + 평가
```

각 하청법인은 서로의 전체 업무를 알지 못하도록 칸막이(compartmentalization)되어 있으며, DUSI ID 체계의 존재 자체를 인지하지 못한다.

### 2-3. 6대 핵심 개념

| 용어 | 의미 | 현실 대응물 |
|---|---|---|
| **박제 (Taxidermy)** | 비정형 현상을 행정 자산으로 고정 | 이상현상의 서류화/자산화 |
| **음각 (Intaglio)** | 구역을 모든 기록에서 삭제 | 지도/등기부에서 구역 소멸 |
| **마모 (Erosion)** | 물리적+행정적 소멸 처리 | 파쇄+서류 삭제 |
| **열화 (Degradation)** | CN 오염도 | 인지 해상도의 붕괴 |
| **행정적 재사회화** | 기억 삭제 + 가짜 인격 주입 | 인간의 공장 초기화 |
| **소급 말소** | 과거까지 소급하여 존재 자체를 삭제 | 완전한 언퍼슨(unperson) |

### 2-4. CN (Cognitive Noise) 체계

비정형 개체가 방출하는 "잔류 노이즈"의 농도를 CN 단위로 측정하며, 인간에 대한 영향이 단계적으로 정의된다.

| CN 수치 | 단계 | 증상 |
|---|---|---|
| 0~10 | 정상 | 없음 |
| 11~40 | 감각 간섭 | 환청(금속 긁는 소리), 환시(시야 가장자리 검은 파편) |
| 41~80 | 인지 마모 | 공간감 상실(직선 복도를 원형으로 인식), 시간 왜곡 |
| 81~150 | 비정형 피폭 | 자아 경계 붕괴, 인프라의 감정을 자기 것으로 느낌 |
| 151+ | 위상적 동화 | 육체가 콘크리트/개체에 물리적으로 융합, 사망 |

"인프라의 감정을 자기 것으로 느낌", "건물 벽면으로 변해간다는 촉각적 환각" 같은 묘사가 이 세계관의 독자적 공포 미학을 보여준다. 단순한 괴물 공포가 아니라 **"도시 인프라 자체가 살아있는 존재"**라는 코즈믹 호러이다.

### 2-5. 언어적 폭력: 행정 순화 용어 대조표

이 세계관에서 가장 섬뜩한 장치 중 하나이다.

| 금지 표현 (현실) | 권장 표현 (행정) |
|---|---|
| "괴물이 울부짖는다" | "비정형 개체가 고주파 음성 데이터를 출력함" |
| "동료가 잡아먹혔다" | "인적 자원이 개체와 위상적으로 통합됨" |
| "지옥 같은 풍경" | "고농도 CN 노출로 인한 환경 해상도 열화" |
| "너무 무섭다" | "인지 부하로 인한 논리적 판단 지연" |
| "피가 쏟아진다" | "고농도 유기 점액질 유출" |

보고서에 "살려주세요"를 쓰면 인지 무결성 훼손의 증거로 간주되어 작성자가 처분 대상이 된다. 공포를 느끼는 것 자체가 행정적 범죄가 되는 구조이다.

### 2-6. 재사회화 페르소나 패키지

기억이 삭제된 요원에게 주입하는 가짜 인생 카탈로그이다.

- **패키지 A "부채 상환자"** (가장 권장): 사채 빚 때문에 공단에 감사하며 노동한다는 설정. 유령지구 경비, 최위험 마모 공정에 배치.
- **패키지 B "속죄하는 고아"**: 부모를 죽게 한 죄책감으로 고통을 감내한다는 설정. 고농도 피폭 청소반에 배치.
- **패키지 C "행정 제일주의자"**: 숫자와 서류 강박증 설정. 물류 매핑, 단말기 점검에 배치.

배치될 보직의 위험도에 따라 "어떤 종류의 고통을 자발적으로 감내하게 만들 것인가"를 설계한다는 점이 이 세계관의 본질적 잔혹함이다.

### 2-7. 사례 문서들이 만드는 내러티브 레이어

양식(서식) 문서만으로는 "설정"에 그치지만, 사례 문서들이 서사적 깊이를 만든다.

#### 음성 로그 복구 기록 — "최 요원"의 마지막 교신

> *"계단이 자꾸 숫자가 바뀐다. 방금은 12칸이었는데, 지금은 한 칸뿐이다."*
> *"벽 저편에서 내가 나한테 말을 거는 거지?"*
> *"내 손이 벽지에 스며들고 있어! [...] 소급 말소하지 마! 제발!"*

이 요원은 물리적 공포뿐 아니라 행정적 말소에 대한 공포를 동시에 느끼고 있다. 죽음보다 "존재했다는 기록 자체가 사라지는 것"을 더 두려워하는 것이 이 세계관 특유의 공포이다.

#### 악성 민원인 처리 사례 — "박 모 씨"

> "밤마다 아파트 벽면이 숨을 쉰다"고 신고한 시민 → 무료 건강검진으로 유인 → 기억 삭제 후 "패키지 F" 주입 → **본인이 신고하던 그 벽 앞에서 매일 8시간씩 균열 측정 업무 배치**

잔인한 아이러니가 사무적 문체로 기록된다는 것이 이 세계관의 핵심 톤이다.

#### 작전 실패 기록 — 제07구역 전원 마모

> 지식 흡수형 개체(`ENTITY-L-003`)가 요원들의 단말기 텍스트를 실시간 변조 → 요원들이 서로를 비정형 개체로 인식하여 교전 → CN 300 돌파 → 구역 전체가 현실에서 분리, 음각 처리

개체가 "행정 시스템(텍스트) 자체"에 간섭할 수 있다는 설정은, 이 세계관이 관료제의 도구(문서, 데이터, 해시)를 공포의 매개체로 전용하고 있음을 보여준다.

### 2-8. 해시 검증 시스템 — "행정적 진실"의 물리학

(주)가온정밀시스템(7000)이 운용하는 SHA-256 해시 검증은 단순한 데이터 무결성 체크가 아니라, **"현실의 진위를 행정이 결정한다"**는 세계관의 존재론적 장치이다.

- 해시가 일치하면 = 행정적 진실
- 해시가 불일치하면 = 비정형 오염이거나, 존재해서는 안 되는 것
- 소급 말소된 사람의 흔적 = 해시값만 남은 공백

"진실"이 물리적 사실이 아니라 전산 시스템의 검증 결과에 의해 결정된다는 설정은, 현대 관료제에 대한 날카로운 풍자이기도 하다.

## Part 3. 종합 평가

### 강점

1. **내부 일관성**: 77개 문서가 상호 참조(`[건설-5201]`, `[의료-6402]` 등)하며 완벽한 교차 참조 체계를 구축한다. 어느 문서에서 시작하든 다른 문서로 자연스럽게 연결된다.

2. **형식이 곧 내용**: 문서의 양식 자체(이중 ID, 순화 용어, 해시 경고문)가 세계관의 억압적 본질을 체현한다. 설정을 "설명"하지 않고 "보여주는" 방식이다.

3. **공포의 이중 구조**: 비정형 개체(코즈믹 호러) + 관료 시스템(행정 호러)이 이중으로 작동한다. 독자는 "괴물"보다 "괴물을 관리하는 시스템"에서 더 큰 공포를 느끼게 된다.

4. **한국적 맥락의 활용**: 공기업-하청 구조, 관공서 문서 양식, 행정 순화어, 민원 대응 매뉴얼 등 한국 사회에서 익숙한 형식을 차용하여 "실제로 있을 법한" 느낌을 극대화한다.

5. **서식과 사례의 쌍 구조**: 빈 양식 → 작성 사례를 나란히 배치하여, "이 시스템이 실제로 가동되고 있다"는 느낌을 준다.

### 독자적 정체성

이 세계관은 **"한국 관료제 공포"**라는 독자적 장르를 구축하고 있다. "공포를 공포라고 말하면 처벌받는" 시스템, 죽음보다 "소급 말소"를 두려워하는 존재론, 피해자가 가해 시스템의 부품으로 재조립되는 순환 구조가 이 세계관만의 핵심 정체성이다.

| 항목 | 설명 |
|---|---|
| 조직 성격 | 한국 정부 산하 공기업 + 하청 법인 구조 |
| 대상 처리 철학 | 박제(자산화) / 음각(기록 소멸) / 마모(물리적 소멸) |
| 인력 관점 | 소모성 자재로 취급. 기억 삭제 후 재배치의 순환 |
| 공포의 원천 | 개체 자체가 아닌, 그것을 관리하는 시스템의 잔혹함 |
| 문서 톤 | 관공서 행정 문서의 건조한 사무체 |
| 고유 장르 | 코즈믹 호러 + 행정 호러의 결합 |
